MicroPython Learning Report

Posted on 2020-07-22

Made by Zhang Wengyu, Liu Jinbo & Chen Derun

1.MicroPython introduction

MicroPython is a programming language based on Python 3, it can run in a limited environment e.g. Microcontroller. MicroPython is designed to be as compatible as possible with Python. Similar with Python, MicroPython also encapsulates a large number of libraries, so it is easy to control LED, button, motor, AD/DA and other sensors in MicroPython by importing functions.

2.Microcontroller for MicroPython

MicroPython is supporting 5 platforms empoldered by 01Studio so far, which are STM32, ESP8266, ESP32, CC3200, K210. What is the most common and basic one is STM32(its specific model number is STM32F405RGT6) that mainly encodes PyBoard v1.1-CN version.

01Studio community: www.01Studio.org
